# Method API Description
1 GET /api/v1/districts/ List and Create district.
2 GET /api/v1/districts/<int:dist_id>/ Retrieve, Update and Destroy district.
3 GET /api/v1/districts/weather/temperature/ Returns Temperature at 2PM for 7 days 64 districts.
4 GET /api/v1/districts/weather/cool-ten/ Returns Top Ten Cool Districts Based on Weather for an average of Next 7 Days at 2PM for 64 Districts.
5 GET /api/v1/districts/travel/cool/?from_id=1&to_id=60&date=2023-11-12 Takes Current and Desired Location with date and Returns the Desired is Cooler or Not.
6 GET /api/v1/districts/travel/cool/rt/?from_id=1&to_id=60&date=2023-11-12 Takes Current and Desired Location with date and Returns the Desired is Cooler or Not. Without caching.
